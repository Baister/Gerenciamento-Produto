# Gerenciamento-Produto

- O programa tem a possibilidade de cadastrar e listar. (02/04/2026)
## O que foi feito até o momento
- O programa foi criado com o objetivo de servir como um sistema. Ou seja, seguindo o CRUD(Criar, listar, atualizar e deletar).
- Para ele ser feito, foi necessário criar uma classe específica para o objeto "Produto", um "SistemaProdutos" feito para gerenciar as funções do CRUD, e para a Main foi feito o menu.
